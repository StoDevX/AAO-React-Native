**Tech Stack:** Expo, React Native, React, CocoaPods, mise tasks, TypeScript, Jest + react-test-renderer.

## Task 6: Module peer-dep sweep

**Files:** 28 `modules/*/package.json` files.

Target deps per module (only bump a field if the module currently lists it):
- `"react"` peer → `"^19.1.0"` (present in 25 modules)
- `"react-native"` peer → `"^0.81.0"` (present in 25 modules)
- `"expo-web-browser"` peer in `modules/open-url` → the exact version string captured in Task 2 Step 3

- [ ] **Step 1: Run the sweep script**

Create a temporary file `scripts/_sdk54-peer-bump.sh` (do not commit):
```bash
#!/usr/bin/env bash
set -euo pipefail

EWB_VERSION="$(jq -r '.dependencies["expo-web-browser"]' package.json)"

for f in modules/*/package.json; do
  tmp=$(mktemp)
  jq \
    --arg ewb "$EWB_VERSION" \
    'if .peerDependencies then
       .peerDependencies |= (
         (if has("react") then .["react"] = "^19.1.0" else . end)
         | (if has("react-native") then .["react-native"] = "^0.81.0" else . end)
         | (if has("expo-web-browser") then .["expo-web-browser"] = $ewb else . end)
       )
     else . end' "$f" > "$tmp" && mv "$tmp" "$f"
done
```
Run:
```sh
chmod +x scripts/_sdk54-peer-bump.sh
./scripts/_sdk54-peer-bump.sh
```

- [ ] **Step 2: Verify the sweep touched only peer-dep fields**

Run: `git diff --stat modules/`
Expected: 28 files changed (or close — depends on which modules have which peers). Check `git diff modules/` and confirm only `peerDependencies` blocks changed.

- [ ] **Step 3: Verify `open-url` got the new `expo-web-browser` version**

Expected: matches the version captured in Task 2 Step 3.
